ステップ1 — 画面に文字を出す（echo）

echo（エコー）は「画面に出す」命令です。

  • 出したい文字は " "（ダブルクオート）で囲む
  • 文の終わりに ;（セミコロン）を付ける（文の終わりの「。」のようなもの。忘れると動きません）
  • // から行末はメモ（コメント）。コンピュータは読み飛ばします
    •

    やってみよう：" " の中の ここを書きかえる を消して、旅に出る と打ってみよう。採点は文字の中身が同じかどうかで決まります（文の最初と最後の空白は気にしません）。

    ステップ2 — 名前をつけて覚えさせる（変数）

    同じ「勇者」を何回も打つの、ちょっと面倒だなあ。

    📝 echo は自動で改行しません。だから2つの文がそのままつながって出ます（改行のやり方は、あとのレッスンで習います）。

    そこで変数。一度だけ名前をつけて覚えさせれば、何度でも呼び出せます。
  • $name … 変数は $（ドル）で始める。これが「覚えさせる名前」
  • = … 右にあるものを、左の名前に入れる合図（読むときは右→左）。「"勇者" を $name に入れる」。算数のイコール（左右が等しい）ではありません
  • .（ドット） … 言葉と言葉をのりでくっつけるイメージ。「アレン」＋「の冒険」→「アレンの冒険」。小数点や句点「。」とは別物です
    •

    まずは値を変えてみよう：下の $name の "勇者" を、あなたの好きな名前に書き換えて ▶ 実行。表示が変わります。（この問題は採点なし。正解マークは出ませんが、下に文字が出ればOK！）

    （ここからは、例として名前を アレン として進めます。）

    クエスト：$name に アレン を入れて、アレンの冒険が始まる！ と表示しよう。

    ステップ3 — 数とステータス、そして連結

    名前のような文字は " " で囲みますが、HPやレベルのような数は囲みません。覚えるのはこれだけ：文字は囲む、数は囲まない。

    変数の良いところは、あとから中身を変えられること。レベルアップしてみましょう。

    同じ $level に新しい数を入れると、古い中身（1）は消えて、新しい中身（5）に入れ替わります。名前（$level）はそのまま、中身だけ交換です。

  • . はいくつでもつなげる（2個でも4個でも同じ）
  • " HP" のように、" " の中にはスペースや記号もそのまま入れられる
    •

    クエスト：下のコードの最後の $name を $hp に直して、アレン HP20 と表示しよう。（スペースや記号はもう入っています。直すのは変数1つだけ！）

    ② じつは、実際のアプリも同じ ⚔️

    ここからは現実のアプリの話。変数はゲームだけのものではありません。買い物アプリでも、商品名や値段を変数に覚えさせて使います。題材が違うだけで、仕組みはまったく同じです。

    まとめ ＆ 仕上げのクエスト

  • 画面に出すのは echo "文字";（最後に ;）
  • 変数は $名前 = 値; で覚えさせ、何度でも呼び出せる・あとで変えられる（文字は " "、数はそのまま）
  • 文字をつなぐのは .
    •

    仕上げのクエスト：下のコードの最後の $name を $level に直して、アレン (レベル5) と表示しよう。（かっこやスペースはもう入っています。直すのは変数1つだけ！）
